# Keyboard & Mouse Bindings Reference

Complete reference of all keyboard shortcuts and mouse interactions in Listen2MeRefined.

---

| `Ctrl+Q` | Stop |
| `Ctrl+Right` | Next track |
| `Ctrl+Left` | Previous track |
| `Ctrl+S` | Shuffle playlist |

### Seeking

| Key | Action |
|---|---|
| `Shift+Right` | Seek forward 5 seconds |
| `Shift+Left` | Seek backward 5 seconds |
| `Ctrl+Shift+Right` | Seek forward 30 seconds |
| `Ctrl+Shift+Left` | Seek backward 30 seconds |

### Volume

| Key | Action |
|---|---|
| `Up` | Volume up +5% (not when a list or text field is focused) |
| `Down` | Volume down -5% (not when a list or text field is focused) |
| `M` | Toggle mute (not when typing in a text field) |
| `Mouse Wheel` (on volume slider) | Volume Â±2% per tick |

### Navigation

| Key | Action |
|---|---|
| `Ctrl+F` | Focus the quick search bar |
| `Ctrl+Shift+F` | Open Advanced Search window |
| `Escape` | Unfocus search bar (when it has focus) |
| `Ctrl+1` | Switch to Playlist view |
| `Ctrl+2` | Switch to Search Results view |

### Playlist Management

| Key | Action |
|---|---|

## Playlist (song list)

| Key | Action |
|---|---|
| `Enter` | Play selected track |
| `Ctrl+J` | Scroll to and select currently playing track |
| `Ctrl+N` | Set selected track as next |
| `Ctrl+Delete` | Remove selected from playlist |
| `Ctrl+Shift+Delete` | Remove selected from playlist |

| Key | Action |
|---|---|
| `F2` | Rename selected playlist |
| `Enter` (while renaming) | Confirm rename |
| `Escape` (while renaming) | Cancel rename |

## Settings Window

### Tab Navigation

| Key | Action |
|---|---|
| `Ctrl+1` | General tab |
| `Ctrl+2` | Playback tab |
| `Ctrl+3` | Library tab |
| `Ctrl+4` | Playlists tab |
| `Ctrl+5` | Hooks & Alerts tab |
| `Ctrl+6` | Advanced tab |
| `Ctrl+Tab` | Next tab |
| `Ctrl+Shift+Tab` | Previous tab |
| `Escape` | Close Settings window |

### Slider Controls

| Input | Action |
|---|---|
| `Mouse Wheel` (on any settings slider) | Adjust value by one tick |

---

## Advanced Search Window

| Key | Action |
|---|---|
| `Enter` (in input field) | Add filter criteria |
| `Ctrl+Enter` | Execute search |
| `Delete` (in criteria list) | Remove selected criteria |
| `Escape` | Close window |

All controls (combo boxes, text input, buttons, radio buttons) are reachable via `Tab`.

---

## Folder Browser

| Key | Action |
|---|---|
| `Enter` or `Right` | Open selected folder |
| `Left` or `Backspace` | Go to parent folder |
| `Ctrl+V` | Jump to clipboard path |
| `Escape` | Cancel and close |

---

## Global Hotkeys (system-wide, work even when app is not focused)

| Key | Action |
|---|---|
| `Ctrl+Alt+N` | Toggle corner "Now Playing" window |
| `Media Play` | Play / Pause |
| `Media Next` | Next track |
| `Media Previous` | Previous track |
| `Media Stop` | Stop |

Global media keys can be toggled in Settings > Hooks & Alerts.
The corner window toggle (`Ctrl+Alt+N`) is always active regardless of settings.

---

## Mouse Interactions

| Input | Location | Action |
|---|---|---|
| Mouse Wheel | Volume slider | Adjust volume |
| Mouse Wheel | Any settings slider | Adjust slider value |
| Double-click | Playlist song | Play song |
| Double-click | Playlist sidebar item | Rename playlist |
| Right-click | Song in list | Open context menu |
| Double-click | Folder in browser | Navigate into folder |
| Mouse to screen corner | Anywhere | Show corner "Now Playing" popup (if enabled in settings) |
